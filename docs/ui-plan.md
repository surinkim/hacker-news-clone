# Hacker News 클론 UI 기획서

## UI 구성 요소
1. **헤더**
   - 사이트 로고
   - 네비게이션 링크 (new, past)

2. **뉴스 목록**
   - 뉴스 항목의 순번
   - 뉴스 제목 (클릭 시 원본 링크로 이동)
   - 뉴스 내용 요약 (3~4줄)

## 페이지 구성
1. **홈 페이지**
   - **헤더**: 상단에 고정된 사이트 로고와 네비게이션 링크
   - **뉴스 목록**: 뉴스 항목의 순번, 제목, 요약된 내용 표시
     - 뉴스 제목 클릭 시 새 탭에서 원본 링크로 이동

## 스타일 가이드
- **폰트**: 기본 폰트는 Arial, sans-serif
- **색상**
  - 배경색: #f6f6ef
  - 텍스트 색상: #333
  - 링크 색상: #0000ff
- **레이아웃**
  - 최대 너비: 800px
  - 중앙 정렬
  - 적절한 여백과 패딩 사용
